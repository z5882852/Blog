以下命令的区别

  • hadoop fs：通用的文件系统命令，针对任何系统，比如本地文件、HDFS文件、HFTP文件、S3文件系统等。
  • hadoop dfs：特定针对HDFS的文件系统的相关操作，但是已经不推荐使用。
  • hdfs dfs：与hadoop dfs类似，同样是针对HDFS文件系统的操作，替代hadoop dfs
